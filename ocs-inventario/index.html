¿Qué vas a aprender?

Todo lo que necesitás para implementar y mantener un sistema de inventario IT centralizado con OCS.

✓
Instalar y configurar el servidor OCS Inventory desde cero
✓
Desplegar el agente OCS en Linux (CentOS/RHEL y Ubuntu/Debian)
✓
Desplegar el agente en Windows y dispositivos Android
✓
Automatizar la instalación del agente Windows por política de dominio (GPO)

Recorrer todas las opciones del panel de administración de OCS
