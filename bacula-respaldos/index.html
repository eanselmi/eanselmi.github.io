¿Qué vas a aprender?

Todo lo que necesitás para implementar y gestionar una solución de backups empresarial con Bacula.

✓
Instalar y configurar Bacula desde cero con laboratorio propio
✓
Agregar clientes Linux (NRPE) y Windows (NSClient) a Bacula
✓
Ejecutar backups y restores completos, diferenciales e incrementales
✓
Administrar con BCONSOLE y Bacula Administration Tool (BAT)

Enviar reportes por email con SMTP-CLI y Postfix

Administrar cintas: carga, descarga, etiquetado y pools

Gestión gráfica avanzada con Baculum

Backup en la nube: crear bucket S3 en AWS y ejecutar backups remotos
